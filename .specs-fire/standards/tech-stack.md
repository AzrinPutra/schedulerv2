# Tech Stack

## Runtime & Language
- **Language**: Python 3.11+
- **Runtime**: CPython

## CLI & TUI Frameworks
- **CLI**: Typer (for command-line interface)
- **TUI**: Rich (for aesthetic terminal UI)

## Data Storage
- **Database**: SQLite (file-based, serverless)

## Testing
- **Unit Tests**: pytest
- **Type Checking**: mypy (optional, enabled for critical paths)

## Development Tools
- **Package Manager**: pip (with `requirements.txt`)
- **Formatter**: Black
- **Linter**: ruff

## CI/CD
- GitHub Actions
- Python version: 3.11, 3.12
